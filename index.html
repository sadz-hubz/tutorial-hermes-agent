🤔 APA ITU HERMES AGENT?

Bayangkan punya asisten pribadi yang bisa jalan di HP Android.

Asisten ini bisa:

  • 📁 Baca & tulis file
  • 🌐 Cari info di internet
  • 💬 Kirim notifikasi ke Telegram
  • ⏰ Ngerjain tugas otomatis (jadwal, monitor)
  • 💡 Ngobrol kayak ChatGPT, tapi punya AKSES ke HP kamu
💡 Hermes Agent BUKAN aplikasi. Dia jalan di terminal (layar hitang). Tapi jangan takut — tinggal copy-paste!

Cara Kerja: HP Android → Termux → Hermes → OpenRouter → AI Model 🚀

📱 TERMUX ITU APA?

Termux = "Windows/Linux mini" di HP Android. Di sinilah Hermes Agent tinggal.

Analogi Sederhana:

🏠 HP Android
Rumah tempat semua tinggal
🚪 Termux
Kamar kerja di dalam rumah
🤖 Hermes
Asisten yang ada di kamar kerja
🧠 OpenRouter
Otak AI yang diakses lewat internet
⚠️ JANGAN install Termux dari Play Store! Pakai F-Droid: f-droid.org/packages/com.termux

📥 INSTALL TERMUX

1

Download Termux dari F-Droid

Buka browser HP, ketik alamat ini:

https://f-droid.org/repo/com.termux_1002.apk
⚠️ Gratis, aman, no iklan. Jangan dari Play Store!
2

Install Termux

Klik Download → Buka file .apk → Klik Install

⚠️ Izinkan "Install from unknown sources" kalau diminta
3

Buka Termux

Buka aplikasinya. Tunggu sampai muncul tanda $

✅ Tanda "$" = Termux siap menerima perintah!

⚙️ PERSIAPAN: UPDATE & INSTALL

Sebelum install Hermes, "beresin kamar kerja" dulu:

1

Update Termux

pkg update && pkg upgrade -y
Tunggu sampai selesai. Kalau ada prompt, ketik "Y" lalu Enter
2

Install Packages Wajib

pkg install -y nodejs python git curl wget nano openssl-tool
Node.js untuk Hermes. Python untuk tools tambahan. Tunggu 1-2 menit
💡 Semua perintah tinggal COPY-PASTE ke Termux. Jangan ketik manual!

🔧 INSTALL HERMES AGENT

1

Quick Install

curl -fsSL https://raw.githubusercontent.com/NousResearch/hermes-agent/main/scripts/install.sh | bash
⏱️ Tunggu 2-5 menit — JANGAN ditutup!
2

Cek Version

hermes --version
✅ Harus muncul: hermes v0.x.x atau serupa
3

Inisialisasi

hermes init
Folder ~/.hermes/ akan terbuat otomatis

🔑 DAPATIN API KEY OPENROUTER

Hermes butuh "kunci" untuk terhubung ke AI. Kunci ini = API Key.

Analogi: API Key itu seperti nomor HP — untuk terhubung ke server AI.

1

Daftar di OpenRouter

Tekan: https://openrouter.ai
Login pakai Google atau GitHub. Gratis, no credit card!
2

Buat API Key

Klik foto profil → "Keys" → "Create Key" → Beri nama: "termux-hermes"

⚠️ COPY key-nya — format: sk-or-v1-xxxxx...
🆓 OpenRouter kasih FREE credits pas daftar! Tanpa kartu kredit!

🆓 FREE MODELS — TANPA BAYAR!

openrouter/auto
Otomatis pilih model gratis terbaik
⭐ RECOMMENDED
gemini-2.0-flash-free
By Google. Cepat & responsif
⚡ CEPAT

⚙️ KONFIGURASI HERMES + OPENROUTER

Sambungkan Hermes ke OpenRouter pakai API Key. Copy-paste perintah ini:

# 1. Setup hermes setup # 2. Select Provider Pilih OpenRouter # 3. Set API Key (GANTI key_kamu dengan key asli!) Paste/Masukan API key OpenRouter # 4. Select default model Cari yang free(openrouter/owl-alpha) # 5. Add anoter credential jika punya lebih dari 1 API key OpenRouter,jika tidak enter saja # 6. Config lanjutan jika tidak mengerti enter-enter saja sampai ke select platform

💬 BONUS: BIKIN BOT TELEGRAM

Mau akses Hermes dari Telegram? Bisa!

1

Buat Bot via @BotFather

Di Telegram, cari @BotFather → /newbot
2

Beri Nama Bot

Nama bebas (contoh: "Hermes Assistant"). Username harus unik & diakhiri "bot"

3

Copy Token

Format: 123456:ABC-DEFxxxxx
4

Set di Hermes

hermes config set telegram.token TOKEN_KAMU
hermes gateway start
💡 Gateway = "jembatan" antara Telegram & Hermes. Harus jalan terus!

🧪 TESTING: PASTIKAN JALAN!

Cek Config Benar:

hermes --version # Harus muncul version number ✅ hermes gateway status # Harusnya "running" ✅

Test Chat:

hermes chat # Ketik "Halo" → Enter → Hermes harus balas! ✅

Test Telegram:

Buka Telegram → Cari bot → /start → Kirim pesan → Harus ada balasan! ✅

💡 Kalau ada error, next slide = troubleshooting!

🔧 TROUBLESHOOTING

"API key invalid" / "401"
Regenerate key di openrouter.ai/keys
"Connection error"
Cek internet / coba lagi nanti
"429 Too Many Requests"
Rate limit → tunggu 1-2 menit
Bot Telegram nggak balas
hermes gateway start
Termux mendadak mati
Battery → Unrestricted

⚡ QUICK START — COPY-PASTE SEMUA

Salin semua perintah ini, paste ke Termux. Tunggu tiap perintah selesai.

# ========= HERMES AGENT QUICK START ========= # 1. Setup Termux (2-3 menit) pkg update && pkg upgrade -y pkg install -y nodejs python git curl wget nano openssl-tool # 2. Install Hermes (2-5 menit) npm install -g @nousresearch/hermes-agent # 3. Init hermes init # 4. Setup OpenRouter (GANTI KEY_KAMU!) hermes config set model.provider openrouter hermes config set model.base_url https://openrouter.ai/api/v1 hermes config set model.api_key sk-or-v1-KEY_KAMU hermes config set model.default openrouter/auto # 5. Test hermes chat # ========= SELESAI! =========
💡KEY_KAMU = API Key dari https://openrouter.ai/keys
